# Bug Reporting & Technical Support

If you encounter any bugs, unexpected behavior, or technical issues while utilizing MADE (v3.0.0), please submit a detailed report to the maintainer.

**Email:** taylosh@iu.edu

**Subject Line:** Bug Report: MADE_v3.0.0

---

## What to Include in Your Report

To facilitate a faster resolution and support the pipeline's robust error-handling design philosophy, please include the following in your email:

### 1. Description of the Error
- What were you trying to extract (which features, which tiers, which filters)?
- What happened instead of the expected behavior?
- Does the error occur consistently or intermittently?

### 2. Terminal Output
Copy and paste any specific error messages or tracebacks displayed in the console. For example:
